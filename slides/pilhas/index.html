Lincoln de Macêdo

O que são pilhas???

  • Pilhas são estruturas de dados que seguem a lógica de que o último a entrar será o primeiro a sair (last in, first out).

  • Alguns exemplos práticos deste tipo de funcionamento que encontramos no dia a dia:

    • Pilha de pratos na pia
    • Pilha de folhas de papel
    • Na hora de limpar um ventilador

Mas e no computador, onde usamos pilhas???

  • Num editor qualquer, quando usamos Ctrl+Z
  • Quando usamos Ctrl+Shift+T para reabrir abas no navegador
  • Alguém consegue pensar em mais um exemplo???

Representando pilhas em linhas de código

O momento mais esperado da aula

Em Java...

import java.util.ArrayList;
public class Pilha {
    private ArrayList elementos = new ArrayList();
    public void setElemento(Integer elm) {
        // adiciona no inicio
        this.elementos.add(0, elm);
    }
    public Integer getElemento() {
        // retira o ultimo adicionado
        Integer elem = this.elementos.remove(0);
        return elem;
    }
    public ArrayList elem(){
        // exibe os elementos guardados
        return this.elementos;
    }
}
